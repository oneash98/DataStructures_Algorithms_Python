### 유클리드 알고리즘

최대공약수 구하는 방법

```
두 양의 정수 a, b (a > b)에 대하여
a = bq + r (0 <= r < b)이라 하면,
gcd(a, b) = gcd(b, r)

r = 0 일 때, a, b의 최대공약수는 b
```

즉, a를 b로 나누었을 때 나머지가 r일 때, a와 b의 최대공약수는 b와 r의 최대공약수와 같음.

b와 r의 최대공약수를 구하기 위해 b를 도 r로 나누고... 반복...

<br/>

예시) 48과 18의 GCD
```
48 / 18 -> 몫 2 나머지 12
18 / 12 -> 몫 1 나머지 6
12 / 6  -> 몫 2 나머지 0

따라서 GCD는 6
```
